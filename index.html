How to Install

This project is a Chrome Extension. You cannot run it like a website (npm start won't work).

  1. Open Chrome.
  2. Navigate to chrome://extensions in the address bar.
  3. Turn on Developer mode using the switch in the top-right corner.
  4. Click the Load unpacked button that appears.
  5. Select the folder containing this index.html file.
  6. Go to Google AI Studio.
Success! You should see a "RTL Mode" button in the bottom-right corner of Google AI Studio.
